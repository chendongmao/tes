# 原始数据共享API

## POST 获取历史数据

POST /iot3/rest/api/v1/history.json

历史数据api

> Body 请求参数

```json
{
  "codes": [
    "C020015H168AAH001001",
    "C02001SW141AAH001001"
  ],
  "stime": 1725120000000,
  "etime": 1727366400000
}
```

### 请求参数

|名称|位置|类型|必选|中文名|说明|
|---|---|---|---|---|---|
|body|body|object| 否 ||none|
|» codes|body|[string]| 是 | 编码集合|none|
|» stime|body|integer| 是 | 开始时间（毫秒）|none|
|» etime|body|integer| 是 | 结束时间（毫秒）|none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "data": [
    {
      "code": "string",
      "time": "string",
      "status": null,
      "value": 0,
      "name": "string",
      "unit": "string"
    }
  ]
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none|状态码|0 表示成功|
|» data|[object]|true|none|数据集合|none|
|»» code|string|true|none|编码|none|
|»» time|string|true|none|时间|none|
|»» status|null|true|none|？|none|
|»» value|integer|true|none|值|none|
|»» name|string|true|none|设备名称|none|
|»» unit|string|true|none|单位|none|

## POST 统计数据

POST /iot3/rest/api/v1/aggregate.json

> Body 请求参数

```json
{
  "codes": [
    "25",
    "18",
    "41"
  ],
  "stime": 1480282605070,
  "etime": 1448940293570
}
```

### 请求参数

|名称|位置|类型|必选|中文名|说明|
|---|---|---|---|---|---|
|body|body|object| 否 ||none|
|» measurement|body|string| 是 | 统计时间间隔|5min, 15min, 30min, 60min, day|
|» codes|body|[string]| 是 | 水务编码集合|none|
|» stime|body|integer| 是 | 开始时间（毫秒）|none|
|» etime|body|integer| 是 | 结束时间（毫秒）|none|
|» fields|body|[string]| 是 | 数据项|sum - 累计值 avg - 平均值mxtime - 最大值时间mxvalue - 最大值mntime - 最小值时间mnvalue - 最小值ftime - 首值时间fvalue - 首值ltime - 末值时间lvalue - 末值|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "data": [
    {
      "time": "string",
      "code": "string",
      "mxvalue": 0,
      "mxtime": "string",
      "mnvalue": 0,
      "mntime": "string",
      "ftime": "string",
      "fvalue": 0,
      "ltime": "string",
      "lvalue": 0,
      "avg": 0,
      "sum": 0,
      "name": "string",
      "unit": "string"
    }
  ]
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none|状态码|0表示成功|
|» data|[object]|true|none|数据|none|
|»» time|string|true|none|时间|none|
|»» code|string|true|none|编码|none|
|»» mxvalue|integer|true|none|最大值|none|
|»» mxtime|string|true|none|最大值时间|none|
|»» mnvalue|integer|true|none|最小值|none|
|»» mntime|string|true|none|最小值时间|none|
|»» ftime|string|true|none|首位时间|none|
|»» fvalue|integer|true|none|首位值|none|
|»» ltime|string|true|none|末位时间|none|
|»» lvalue|integer|true|none|末位值|none|
|»» avg|integer|true|none|平均值|none|
|»» sum|integer|true|none|总和|none|
|»» name|string|true|none|设备名称|none|
|»» unit|string|true|none|单位|none|
